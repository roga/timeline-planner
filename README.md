## Timeline Planner

- 簡易時間軸任務管理系統
- 資料可以支援匯入匯出，操作時儲存於 localStorage。
- Tech Stack: React + TailwindCSS + Vite

### 開發與佈署

- 下載並安裝最新 Node.js
- 請在終端機中輸入以下指令，確認 Node.js 與 npm 安裝成功

```
node -v
npm -v
```

- 安裝程式及開發環境

```
git clone https://...
npm install
```

- 啟動開發環境

```
npm run dev 
```

- 用瀏覽器進入

```
http://localhost:5173
```

- 產生可供佈署的 HTML 檔案，會將所有結果輸出至 `dist/` 目錄下

```
npm run build
```
